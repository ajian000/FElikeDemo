# 魔法战棋 Demo

基于火焰纹章风格的魔法战棋游戏演示，实现了策划案中的核心玩法机制。

## 已实现功能

### ✅ 核心系统
- **回合制战斗**: 玩家与敌方交替行动
- **方格战场**: 20x15网格地图，支持不同地形
- **永久死亡**: 单位阵亡后无法复活
- **移动与攻击**: 点击选择单位，使用操作面板进行行动

### ✅ 地图编辑器
- **地形绘制**: 8种地形类型(草地、水域、山地、燃烧、冰冻、石柱、墙壁、地板)
- **单位放置**: 9种单位类型(玩家5种+敌方4种)
- **室内/室外切换**: 支持室内和室外两种地图模式
- **地图保存/加载**: 本地存储和JSON导出
- **应用到游戏**: 一键应用自定义地图到游戏

### ✅ 技能树编辑器
- **三大体系**: 自然魔法(环形)、神圣魔法(三角形)、古代魔法(丰字形)
- **节点管理**: 添加、删除、编辑技能节点
- **连接系统**: 可视化连接线，支持技能前置关系
- **节点类型**: 基础、进阶、高阶、终极、被动、跨系节点
- **属性系统**: 支持火、冰、风、雷、裁决、恩典、救赎、古代等属性
- **交互操作**: 拖拽移动节点、双击编辑、右键删除
- **保存/加载**: 本地存储和JSON导出导入
- **重置功能**: 一键恢复默认技能树

### ✅ 室内/室外地图差异
- **室外地图**: 丰富地形类型(草地、水域、山地、魔法节点等)，支持魔法改变地形，超远程技能可用
- **室内地图**: 简洁地形(地板、墙壁等)，无动态地形变化，**超远程技能(射程≥4)禁用**
- **地形互动**: 室外支持魔法深度干预地形，室内仅支持基础场景互动

### ✅ 三大魔法体系
1. **自然魔法** (红色)
   - 火法师: 火球术、烈焰爆发
   - 冰法师: 冰锥术、冰封路径
   - MP消耗机制

2. **神圣魔法** (黄色)
   - 神官: 治疗术、圣光术、净化
   - 无MP限制，可持续使用
   - 支援友军与攻击敌人

3. **古代魔法** (紫色)
   - 古代法师: 陨石坠落、地壳隆起
   - 咏唱机制: 需要等待1-3回合
   - 超高威力与大范围效果

### ✅ 魔法改变地形机制
- **火焰魔法**: 燃烧地面 (增加移动消耗)
- **冰系魔法**: 冻结水域 (使水面可通行)
- **土系魔法**: 隆起石柱 (完全阻挡移动)

### ✅ 地形系统
- **室外地形**: 草地(正常)、水域(阻挡)、山地(防御+1)、燃烧(消耗+2)、冰冻(可通行)、石柱(阻挡)
- **室内地形**: 地板(正常移动)、墙壁(完全阻挡)

### ✅ 单位类型
- 5个玩家单位: 火法师、冰法师、神官、古代法师、骑士
- 4个敌方单位: 火法师、冰法师、神官、战士

### ✅ AI系统
- 敌方自动寻找目标
- 优先使用魔法攻击
- 自动移动靠近目标
- 咏唱等待机制

### ✅ 技能树体系（策划案实现）
1. **自然魔法体系** (环形结构)
   - **核心层**: 火、冰、风、雷四种元素基础技能
   - **内层**: 各元素专属基础强化(提升伤害、扩大范围等)
   - **中层**: 元素高级技能，需对应内层节点解锁
   - **外层**: 元素高阶核心技能 + 跨系中转功能
   - **跨系方式**: 负面代价节点 + 元素共鸣节点
   - **加点规则**: 无强制路径，相邻节点即可解锁

2. **神圣魔法体系** (三角形结构)
   - **三个角**: 裁决(攻击)、恩典(支援)、救赎(恢复)
   - **基础技能**: 三方向基础技能无门槛解锁
   - **技能强化**: 擅长方向CD减免，其他方向无特权
   - **终极技能**: "三系合一"中心聚合，需解锁两个方向的高阶节点
   - **加点规则**: 距离基础节点越远，技能点消耗越多

3. **古代魔法体系** (丰字形结构)
   - **核心主线**: 中间纵向路径，威力几何级增长，咏唱回合同步增加
   - **消耗倍增**: 主线技能技能点消耗成倍增加(前一个的2倍)
   - **辅助支线**: 两侧横向路径，降低咏唱风险
   - **支线特点**: 固定低成本，需先解锁对应主线节点
   - **加点规则**: 主线递进+支线选择，简洁聚焦

## 操作说明

### 基本操作
1. **选择单位**: 点击己方单位(左侧彩色圆圈)进行选择
2. **移动**: 点击"移动"按钮，显示绿色移动范围，点击目标格子移动
3. **攻击**: 点击"攻击/魔法"按钮，显示红色攻击范围，点击敌人选择魔法
4. **待机**: 点击"待机"按钮结束单位当前回合
5. **结束回合**: 完成所有单位操作后，点击"结束回合"

### 地图编辑器
1. **打开编辑器**: 点击顶部"🗺️ 地图编辑器"按钮
2. **选择工具**: 左侧工具栏选择地形或单位类型
3. **绘制地形**: 左键拖动绘制地形，右键擦除单位
4. **切换模式**: 通过下拉菜单切换室内/室外地图模式
5. **保存地图**: 点击"保存地图"保存到本地存储
6. **加载地图**: 点击"加载地图"从本地存储读取
7. **导出JSON**: 点击"导出JSON"下载地图文件
8. **应用游戏**: 点击"应用到游戏"将自定义地图加载到游戏中

### 魔法使用
- 选择单位后，右侧面板会显示可用魔法列表
- 点击"攻击/魔法"后选择目标
- 在弹出框中输入魔法序号(1, 2, 3...)选择魔法
- 神圣魔法(神官)可以无限制使用基础魔法
- **室内地图禁用射程≥4的超远程技能**

### 地形互动
- 火法师的"火球术"可以燃烧地面
- 冰法师的"冰封路径"可以冻结水域
- 古代法师的"地壳隆起"可以升起石柱
- 魔法攻击到目标时也会改变该格子地形

### 技能树编辑器使用
1. **打开编辑器**: 点击游戏顶部"⚡ 技能树编辑器"按钮
2. **选择体系**: 左侧选择自然魔法/神圣魔法/古代魔法
3. **工具选择**:
   - 选择/查看: 查看节点属性，拖拽移动位置
   - 添加节点: 点击空白处添加新技能节点
   - 删除节点: 点击节点删除（会删除相关连接）
   - 连接节点: 依次点击两个节点创建连接
4. **编辑节点**: 双击节点或点击右侧"编辑节点"按钮
5. **节点属性**:
   - 技能名称、类型、属性
   - 所需技能点、技能效果
   - 附加属性(跨系解锁、负面效果、核心技能)
6. **保存操作**:
   - 保存技能树: 保存到本地存储
   - 加载技能树: 从本地存储读取
   - 导出JSON: 下载技能树文件
   - 重置技能树: 恢复默认状态

### 技能树结构说明
- **自然魔法(环形)**: 核心层→内层→中层→外层，四元素独立分布
- **神圣魔法(三角形)**: 三个角的基础技能向中心聚合，汇聚到终极技能
- **古代魔法(丰字形)**: 纵向核心主线+两侧横向支线，简洁聚焦
- **节点颜色**: 绿色(基础) → 蓝色(进阶) → 紫色(高阶) → 金色(终极) → 灰色(被动) → 橙色(跨系)
- **核心标记**: 终极技能显示⭐标记
- **跨系标记**: 跨系节点显示🔄标记

### 咏唱机制
- 古代法师的强力魔法需要咏唱
- 咏唱期间单位显示✧标记
- 咏唱完成后自动释放魔法
- 咏唱期间可以被打断(单位阵亡)

## 游戏规则

- **永久死亡**: 单位HP归零后永久从队伍中移除
- **回合限制**: 每个单位每回合只能移动一次、攻击一次
- **MP恢复**: 玩家回合开始时，魔法师恢复5点MP(神圣魔法除外)
- **地形效果**: 不同地形提供不同的防御加成和移动消耗
- **室内限制**: 室内地图禁止使用射程≥4的超远程技能
- **克制关系**:
  - 自然魔法克制古代魔法(快速输出打断咏唱)
  - 古代魔法克制神圣魔法(高额伤害突破续航)
  - 神圣魔法克制自然魔法(持续支援压制均衡输出)
