# 魔法战棋 Demo - 快速使用指南

## 🎮 游戏操作

### 基础玩法
1. **选择单位**: 点击左侧彩色圆圈(己方单位)
2. **移动**: 点击"移动" → 选择绿色格子
3. **攻击**: 点击"攻击/魔法" → 选择红色范围内的敌人 → 输入魔法序号
4. **待机**: 完成行动后点击"待机"
5. **结束回合**: 所有单位行动完毕后点击"结束回合"

### 单位说明
- 🔴 **火法师**: 高伤害自然魔法，可燃烧地形
- 🔵 **冰法师**: 控制型，可冻结水域
- 🟡 **神官**: 神圣魔法，治疗+攻击(无MP限制)
- 🟣 **古代法师**: 超高伤害，需咏唱1-2回合
- 🔷 **骑士**: 物理单位，高血量高防御

### 魔法示例
```
火法师:
  1. 火球术(MP:5 威力:12) - 燃烧目标地面
  2. 烈焰爆发(MP:8 威力:18) - 高伤害

神官:
  1. 治疗术(MP:0 威力:-15) - 恢复友军生命
  2. 圣光术(MP:0 威力:8) - 神圣伤害
  3. 净化(MP:0) - 解除负面状态

古代法师:
  1. 陨石坠落(MP:15 威力:25 咏唱:2回合)
  2. 地壳隆起(MP:12 威力:10 咏唱:1回合) - 升起石柱
```

## 🗺️ 地图编辑器

### 打开编辑器
点击游戏顶部 **"🗺️ 地图编辑器"** 按钮

### 地形工具
- 🌿 **草地**: 正常移动
- 💧 **水域**: 阻挡移动(室外)
- ⛰️ **山地**: 防御+1，移动消耗+2(室外)
- 🔥 **燃烧**: 移动消耗+2
- ❄️ **冰冻**: 水域可通行
- 🪨 **石柱**: 完全阻挡
- 🧱 **墙壁**: 室内障碍(室内)
- ⬜ **地板**: 室内地面(室内)

### 单位工具
- **玩家单位**: 火法师、冰法师、神官、古代法师、骑士
- **敌方单位**: 火法师、冰法师、神官、战士

### 编辑操作
- **左键绘制**: 绘制地形或放置单位
- **右键擦除**: 擦除单位
- **拖动绘制**: 按住左键拖动连续绘制

### 地图设置
- **地图类型**: 下拉菜单选择"室外地图"或"室内地图"
- **清空地图**: 重置为空地图
- **填充草地**: 填充基础地形

### 保存与加载
- **保存地图**: 保存到浏览器本地存储
- **加载地图**: 从本地存储读取
- **导出JSON**: 下载地图文件(可分享给他人)
- **应用到游戏**: 将自定义地图加载到游戏中并开始战斗

## ⚡ 技能树编辑器

### 打开编辑器
点击游戏顶部 **"⚡ 技能树编辑器"** 按钮

### 三大魔法体系
1. **🌿 自然魔法** (环形结构)
   - 四元素核心层(火、冰、风、雷)
   - 内层: 元素强化
   - 中层: 高级技能
   - 外层: 终极技能+跨系节点

2. **✨ 神圣魔法** (三角形结构)
   - 三个角: 裁决、恩典、救赎
   - 向中心汇聚到终极技能"三位一体"

3. **🔮 古代魔法** (丰字形结构)
   - 核心主线: 咏唱技能，威力递增
   - 两侧支线: 被动技能，降低风险

### 编辑工具
- **👆 选择/查看**: 选择节点查看属性，拖拽移动位置
- **➕ 添加节点**: 点击空白处添加新技能节点
- **🗑️ 删除节点**: 点击节点删除(会删除相关连接)
- **🔗 连接节点**: 依次点击两个节点创建连接线

### 节点编辑
1. **双击节点** 或点击右侧"编辑节点"按钮
2. **填写属性**:
   - 技能名称、类型、属性
   - 所需技能点、技能效果
   - 附加属性(跨系解锁、负面效果、核心技能)
3. **保存修改**

### 节点类型颜色
- 🟢 绿色: 基础技能
- 🔵 蓝色: 进阶技能
- 🟣 紫色: 高阶技能
- 🟡 金色: 终极技能 (⭐标记)
- ⚫ 灰色: 被动技能
- 🟠 橙色: 跨系节点 (🔄标记)

### 技能树操作
- **💾 保存技能树**: 保存到本地存储
- **📂 加载技能树**: 从本地存储读取
- **📤 导出JSON**: 下载技能树文件
- **🔄 重置技能树**: 恢复默认状态

### 技能树加点规则
- **自然魔法**: 无强制路径，相邻节点即可解锁
- **神圣魔法**: 距离基础节点越远，技能点消耗越多
- **古代魔法**: 主线技能消耗成倍增加，支线固定成本低

## 🏠 室内/室外地图差异

### 室外地图
- ✅ 丰富地形类型
- ✅ 支持魔法改变地形
- ✅ 超远程技能(射程≥4)可用
- 📌 适合大规模战斗

### 室内地图
- ⬜ 简洁地形(地板+墙壁)
- ❌ 无动态地形变化
- ❌ 超远程技能(射程≥4)禁用
- 📌 适合近距离战术

## 💡 战术技巧

### 1. 地形利用
- 利用山地、墙壁的高防御保护脆皮单位
- 用石柱分割敌军阵型
- 冻结水域开辟新路径

### 2. 魔法搭配
- 神官优先治疗，保障续航
- 自然法师快速输出打断敌方咏唱
- 古代法师在安全位置咏唱大招
- 骑士前排抗伤害，保护后排

### 3. 技能树规划
- **自然魔法**: 选择单一元素专精或跨系组合
- **神圣魔法**: 擅长方向CD减免，其他方向成本递增
- **古代魔法**: 主线高成本，支线低成本低风险

### 4. 永久死亡提醒
- 慎重保护神官和古代法师
- 优先击杀敌方高威胁单位
- 利用待机保护残血单位

### 5. 室内战斗
- 超远程技能无法使用，选择近距离魔法
- 利用墙壁作为掩护
- 优先清理狭窄通道的敌人

玩家回合
  ├─ 选择单位

  ├─ 攻击 → 红色范围 → 选择魔法

  ├─ 寻找最近目标

      ↓
回到玩家回合

## ⚙️ 快捷提示

### 游戏提示
- **MP不足**: 检查单位MP是否足够(神官除外)
- **射程不够**: 移动单位靠近目标
- **室内无法使用**: 超远程技能在室内禁用
- **咏唱中**: 单位显示✧标记，等待自动释放
- **地形阻挡**: 水域、石柱、墙壁无法通行

### 编辑器提示
- **地图编辑器**: 左键绘制，右键擦除，拖动连续绘制
- **技能树编辑器**: 双击编辑节点，拖拽移动位置
- **保存位置**: 所有数据保存在浏览器本地存储

## 📚 技能树设计理念

根据策划案实现的核心机制:

### 自然魔法 - 环形结构
- **属性隔离**: 四元素独立分布，无直接通路
- **可控跨系**: 两种方式(负面代价节点+元素共鸣)
- **灵活加点**: 相邻节点即可解锁，无强制路径

### 神圣魔法 - 三角形结构
- **三向专精**: 裁决(攻击)、恩典(支援)、救赎(恢复)
- **成本递增**: 距离基础节点越远，消耗越多
- **终极聚合**: 三系合一"三位一体"终极技能

### 古代魔法 - 丰字形结构
- **主线聚焦**: 核心攻击主线，威力几何级增长
- **消耗倍增**: 主线技能成本成倍增加
- **支线辅助**: 固定低成本，降低咏唱风险
